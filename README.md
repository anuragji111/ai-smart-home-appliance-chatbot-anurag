# AI-Powered Smart Home Appliance Buyer Guide Chatbot

🚀 Live Demo: https://chat-anurag.netlify.app/

---

## 📌 Overview

The **AI-Powered Smart Home Appliance Buyer Guide Chatbot** is an intelligent web-based assistant that helps users choose the best home appliances based on their needs, budget, and usage requirements.

It uses Google's Gemini API to provide real-time AI-generated recommendations, comparisons, and energy-saving suggestions.

---

## 🎯 Project Objective

To develop an AI-powered chatbot that assists users in selecting the most suitable home appliances such as ACs, refrigerators, washing machines, and more by analyzing user input like budget, room size, and preferences.

---

## 💡 Key Features

✔️ Suggests best appliances based on user needs  
✔️ Compares different products  
✔️ Provides detailed feature explanations  
✔️ Gives energy-saving tips  
✔️ Real-time AI responses using Gemini API  
✔️ Simple and clean user interface  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- AI API: Google Gemini API  
- Deployment: Vercel  

---

## ⚙️ Methodology

1. Utilize relevant frontend and backend tools  
2. Fetch responses using Google Gemini API  
3. Process user queries in backend  
4. Generate AI-based recommendations  
5. Display personalized appliance suggestions to users  

---

## 💬 Example Use Case

User Query:
> "Best AC under ₹30,000 for small room?"

Chatbot Response:
✔ Suggests top AC options  
✔ Explains energy efficiency  
✔ Compares brands  
✔ Gives maintenance tips  

---

## 🚀 Setup Instructions

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/google-gemini-chatbot.git
cd google-gemini-chatbot

2️⃣ Install Dependencies
npm install

3️⃣ Environment Setup

Create a .env file in root directory:
GEMINI_API_KEY=your_api_key_here

4️⃣ Run Project:
npm start

```
